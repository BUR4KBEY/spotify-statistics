# Spotify Statistics

Simple application that shows your spotify top tracks/artists.

## Live Demo

[http://stats4spotify.netlify.app](http://stats4spotify.netlify.app)

## Installation

```
yarn install
```

## Setting Up

Rename `.env.example` to `.env` and fill it.

## Starting & Building

-   To Start:
    ```
    yarn start
    ```
-   To Build:
    ```
    yarn build
    ```
